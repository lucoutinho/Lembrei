Lembrei!

Aplicativo mobile desenvolvido em Android Studio com Kotlin, com o objetivo de auxiliar usuários na organização de tarefas e compromissos.

Funcionalidades

- Cadastro de lembretes
- Exibição de lembretes na tela principal
- Armazenamento local com SharedPreferences
- Notificações no dispositivo

Tecnologias utilizadas

- Kotlin
- Android Studio
- XML (Views)
- NotificationManager
- SharedPreferences
